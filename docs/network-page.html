Steam Network

Author

Surya Rao

Last updated

May 6, 2025

For this component, I have displayed the network of the top 10 games with the highest ratings on Metacritic in general and in each cluster from the cluster analysis. Metacritic is a platform that averages critic scores for video games, a lot like Rotten Tomatoes.

Variables

  • For the edges, the variable I used was tags, specifically the number of shared tags between each game. Tags are user-suggested words or short phrases that describe each game. The number of shared tags determines the color and width of each edge.

  • The other variable displayed on the network is a boolean node variable which records if a game’s Steam reviews are overwhelmingly positive. On Steam, a user can rate a game with either a thumbs-up or down. If 95% or more of the reviews are a thumbs up, the game is marked as having overwhelmingly positive reviews. The dataset gives the number of positive user reviews and negative user reviews. I divided positive reviews by the sum of positive user reviews and negative user reviews to get the percentage. The node is colored red if the percentage is below 95%, but blue if it is.

General Top 10 Network

The network shows a few interesting things about the top Metacritic-rated games. One thing I noticed was that action games in famous high-budget series like Grand Theft Auto and The Elder Scrolls had a higher degree of centrality, while games in other genres like Portal 2 and Persona 5 had a much lower degree of centrality. This shows the tendency of these large-budget action games to get high Metacritic scores. The average weighted degree is 1093.8, which is higher than many of the cluster’s average weighted degrees. This shows that the top-rated games on Metacritic are similar in their many ways.

Cluster 2

Thus network has an average weighted degree of 874.2, which is reasonably high when compared to other clusters. Only one of these games is overwhelmingly positive on Steam - Bloons TD 5, this game also has a notably high weighted degree. The game Konung 2 has the lowest weighted degree in this network and is tied for the lowest degree on any network. It seems that most of the top-rated games in the cluster are generally similar, as seen by the average weighted degree, except Konung 2, and are not considered to be excellent games, except for Bloons TD5.

Cluster 3

This network has a higher range of weighted degree than any other cluster, with a weighted degree of 1494.2. Many of these games are reviewed as being overwhelmingly positive. The games in this network are overrepresented in the top 10 games, showing that this type of game (one that has high age requirements, price, and recommendations) is predisposed to succeed critically. The high average weighted degree suggests there is a sort of formula in terms of the types of games that make it into this cluster and get high metacritic ratings.

Cluster 4

This network has a very low weighted degree of 337, the lowest of any cluster. None of these games are reviewed as being overwhelmingly positive. The degree of these nodes varies wildly, with Perimeter 2 having only one edge and a very low weighted degree, while Everreach has a much higher weighted degree than any other game in the network, showing considerable variation in terms of tags for this cluster. This shows that highly rated games from this cluster have not much to do with each other and are not very well liked by the Steam community.

Cluster 5

This network has a reasonably small weighted degree of 753.4. It has by far the most number of games rated Overwhelmingly Positive by the Steam community. Most of these games have a high weighted degree except Team Fortress 2, suggesting it is a bit of an outlier in the top games of this cluster. This network shows that the highest rated games in cluster 5 are very well liked and have diverse play styles, as seen by their comparatively small average weighted degree.

Conclusion

These networks show that games with high Metacritic scores often share many characteristics, demonstrated by shared tags. Some of the clusters in the previous analysis have more shared characteristics with highly reviewed games than others, and some clusters are overrepresented in the top games when compared to the rest. Finding the shared characteristics of these highly rated games, especially those in cluster 3, which are overrepresented in the general top ten network, might be useful for predicting if a game will be a critical success. In the future, I would like to create a network of all of the data and find the shared tags that correlate most to higher critic scores.
